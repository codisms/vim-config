# vim-config

## Installation
```
curl -sSL -H 'Cache-Control: no-cache' https://github.com/codisms/vim-config/raw/master/setup.sh | bash -s [--build]
```

## Notable Options
* Pathogen
* Uses solarized coloring
* Tabstop = 4
* Line numbers and leading space markers visible
* Go fmt command set to goimports
* dbext set to PGSQL by default

## Key Mappings
* Ctrl-h = scroll window left
* Ctrl-l = scroll window right
* Tab = Switch windows
* All arrows keys disabled (purist)

## Command Mappings
* tk = first tab
* tl = next tab
* th = previous tab
* tj = last tab
* tq = close tab
* tc = new tab
* t, = rename tab
* t# = goto tab #
